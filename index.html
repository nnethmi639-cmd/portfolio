About

I'm an HNDIT (Higher National Diploma in Information Technology) student, building complete web and mobile applications as coursework and side projects — usually the kind with a real database schema, real user roles, and real edge cases to debug at 2am.

Most of what's below started as an academic requirement and grew past it: multi-language support nobody asked for, payment gateway sandboxes, CSV exports, PDF invoices. I like finishing the unglamorous 20% of a project as much as the first 80%.

Comfortable moving between a PHP/MySQL backend, a Java/JSP servlet stack, and a Flutter or native Android front end — often within the same week.
